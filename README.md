# nim

### Description

The board comprises five rows of stars:<br>
1: \* \* \* \* \*<br>
2: \* \* \* \*<br>
3: \* \* \*<br>
4: \* \*<br>
5: \* <br>

Two players take turn to remove one or more stars from the end of a single row.

The winner is the player who removes the last star(s) from the board.
